DataReaderActivator - методы

Тип DataReaderActivator предоставляет следующие члены.

Методы
  ИмяОписание
CheckInitialized
Выполняет проверку инициализации.
(Унаследован от InitializableImpl.)
CheckNotInitialized
Выполняет проверку не инициализации.
(Унаследован от InitializableImpl.)
DoInitialize
Инициализация. Настройка группы, установка асинхронного метода чтения данных для указанного тега.
(Переопределяет InitializableImplDoInitialize.)
DoUninitialize
Деинициализация. Закрытие соединения, очистка группы OPC.
(Переопределяет InitializableImplDoUninitialize.)
Equals
Определяет, равен ли заданный объект Object текущему объекту Object.
(Унаследован от Object.)
Finalize
Позволяет объекту Object попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект Object будет утилизирован в процессе сборки мусора.
(Унаследован от Object.)
GetHashCode
Играет роль хэш-функции для определенного типа.
(Унаследован от Object.)
GetType
Возвращает объект Type для текущего экземпляра.
(Унаследован от Object.)
Initialize
Инициализация.
(Унаследован от InitializableImpl.)
IsInitialized
Состоянии инициализации.
(Унаследован от InitializableImpl.)
MemberwiseClone
Создает неполную копию текущего объекта Object.
(Унаследован от Object.)
SetActivationItemName
Установить имя тега, значение которого мониторится. SYS_PV_StatePos.
SetDataReader
Установить DataReader, активируемый в случае начала плавки.
SetOpcConnection
Установить ConnectionHolder для OPC сервера.
ToString
Возвращает объект String, который представляет текущий объект Object.
(Унаследован от Object.)
Uninitialize
Деинициализация.
(Унаследован от InitializableImpl.)
