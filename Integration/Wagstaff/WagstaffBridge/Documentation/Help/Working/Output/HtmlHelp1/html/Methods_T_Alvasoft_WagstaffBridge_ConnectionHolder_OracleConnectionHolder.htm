OracleConnectionHolder - методы

Тип OracleConnectionHolder предоставляет следующие члены.

Методы
  ИмяОписание
CloseConnection
Закрыть подключение.
(Переопределяет ConnectionHolderBaseCloseConnection.)
DangerousConnect
Выполнить опасное подключение. Может завершиться с исключением.
(Переопределяет ConnectionHolderBaseDangerousConnect.)
Equals
Определяет, равен ли заданный объект Object текущему объекту Object.
(Унаследован от Object.)
Finalize
Позволяет объекту Object попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект Object будет утилизирован в процессе сборки мусора.
(Унаследован от Object.)
GetHashCode
Играет роль хэш-функции для определенного типа.
(Унаследован от Object.)
GetHolderName
Вернуть имя теукщей реализации ConnectionHolder.
(Унаследован от ConnectionHolderBase.)
GetOracleConnection
Вернуть OracleConnection.
GetType
Возвращает объект Type для текущего экземпляра.
(Унаследован от Object.)
IsConnected
Подключен ли сервер.
(Переопределяет ConnectionHolderBaseIsConnected.)
LockConnection
Заблокировать подключение.
(Унаследован от ConnectionHolderBase.)
MemberwiseClone
Создает неполную копию текущего объекта Object.
(Унаследован от Object.)
OpenConnection
Подключиться.
(Унаследован от ConnectionHolderBase.)
ProcessError
Обработать ошибку.
(Переопределяет ConnectionHolderBaseProcessError(Exception).)
ReleaseConnection
Разблокировать подключение.
(Унаследован от ConnectionHolderBase.)
SetCallback
Установить Callback.
(Унаследован от ConnectionHolderBase.)
SetCheckConnectionInterval
Установить интервал, через который будет происходить вызов функции проверки подключения.
(Унаследован от ConnectionHolderBase.)
SetHolderName
Установить имя текущей реализации ConnectionHolder.
(Унаследован от ConnectionHolderBase.)
SetLastOperationAllowedTime
Установить максимально позволенный интервал между операциями на сервере, по истечению которого будет происходить проверка подключения.
(Унаследован от ConnectionHolderBase.)
SetReconnectionInterval
Установить интервал, через который будет происходить вызов функции подключения.
(Унаследован от ConnectionHolderBase.)
TestConnectionIsOpen
Тестировать соединение.
(Переопределяет ConnectionHolderBaseTestConnectionIsOpen.)
ToString
Возвращает объект String, который представляет текущий объект Object.
(Унаследован от Object.)
TryReconnect
Выполнить безопасное подключение. Вызывается таймером.
(Унаследован от ConnectionHolderBase.)
UpdateLastOperationTime
Устанавливает метку времени последней операции на текущее время.
(Унаследован от ConnectionHolderBase.)
