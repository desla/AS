NotInitializedException - методы

Тип NotInitializedException предоставляет следующие члены.

Методы
  ИмяОписание
Equals
Определяет, равен ли заданный объект Object текущему объекту Object.
(Унаследован от Object.)
Finalize
Позволяет объекту Object попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект Object будет утилизирован в процессе сборки мусора.
(Унаследован от Object.)
GetBaseException
При переопределении в производном классе возвращает исключение Exception, которое является корневой причиной одного или нескольких исключений.
(Унаследован от Exception.)
GetHashCode
Играет роль хэш-функции для определенного типа.
(Унаследован от Object.)
GetObjectData
При переопределении в производном классе задает сведения об исключении для SerializationInfo.
(Унаследован от Exception.)
GetType
Возвращает тип текущего экземпляра в среде выполнения.
(Унаследован от Exception.)
MemberwiseClone
Создает неполную копию текущего объекта Object.
(Унаследован от Object.)
ToString
Создает и возвращает строковое представление текущего исключения.
(Унаследован от Exception.)
