ConnectionHolderCallback - методы

Тип ConnectionHolderCallback предоставляет следующие члены.

Методы
  ИмяОписание
OnConnected
Вызывается в случае установления соединения с сервером.
OnDisconnected
Вызывается в случае отключения от сервера.
OnError
Вызывается в случае обработки ошибки.
