Программное обеспечение интеграции заводской ИТС с АСУТП Wagstaff
DataBuffer - класс
Буфер для данных из OPC. В случае отсутствия соединения с ИТЦ сохраняет данные в собственную базу. При восстановлении связи перебрасывает данные в ИТЦ.
