DataReader - методы

Тип DataReader предоставляет следующие члены.

Методы
  ИмяОписание
CheckInitialized
Выполняет проверку инициализации.
(Унаследован от InitializableImpl.)
CheckNotInitialized
Выполняет проверку не инициализации.
(Унаследован от InitializableImpl.)
DoInitialize
Инициализация. Сосздание группы, ее настройка.
(Переопределяет InitializableImplDoInitialize.)
DoUninitialize
Деинициализация. Удаление группы тегов.
(Переопределяет InitializableImplDoUninitialize.)
Equals
Определяет, равен ли заданный объект Object текущему объекту Object.
(Унаследован от Object.)
Finalize
Позволяет объекту Object попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект Object будет утилизирован в процессе сборки мусора.
(Унаследован от Object.)
GetHashCode
Играет роль хэш-функции для определенного типа.
(Унаследован от Object.)
GetReadInterval
Получить интервал чтения данных.
GetType
Возвращает объект Type для текущего экземпляра.
(Унаследован от Object.)
Initialize
Инициализация.
(Унаследован от InitializableImpl.)
IsActive
Получить текущее состояние работы.
IsInitialized
Состоянии инициализации.
(Унаследован от InitializableImpl.)
MemberwiseClone
Создает неполную копию текущего объекта Object.
(Унаследован от Object.)
SetActive
Установить состояние работы.
SetCallback
Установить callback, который будет вызываться при получении данных.
SetConfiguration
Установить конфигурацию.
SetData
Установить DataReadInfo, где хранится информация о читаемых данных - идентификаторы, имена.
SetOpcConnectionHolder
Установить ConnectionHolder для OPC сервера.
SetReadInterval
Установить интервал чтения данных.
ToString
Возвращает объект String, который представляет текущий объект Object.
(Унаследован от Object.)

Деинициализация.
(Унаследован от InitializableImpl.)
